
# Vending Machine

A command-line application that simulates a Vending Machine with a twist. It has 2 users with different functionalities each. This application aims to provide an interactive and realistic vending machine experience while demonstrating core programming concepts such as file handling, dynamic memory allocation, modular design, and user authentication.

## Authors

- [@ldkv9439_FatimaKhan](https://github.com/ldkv9439)


## Features

- User Authentication
- Stock Management
- Error Handling
- Dynamic Memory Management


## Run Program

Compile Program by

```bash
  gcc main.c owner. client.c -o main.out
```
